# Gatsby Theme Modern Portfolio

A theme to allow you to simply create a portfolio site

This repo uses yarn workspaces to allow for easier theme [development](https://www.gatsbyjs.org/blog/2019-05-22-setting-up-yarn-workspaces-for-theme-development/).

To use the theme see the README in `gatsby-theme-modern-portfolio`.
Example is in `example`
